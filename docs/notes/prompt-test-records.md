### Research Prompt 1：基础行业信息总结

### 测试输入

```text
你是一个 AI 行业研究分析师。

你的任务是对用户提供的行业信息进行结构化总结，帮助读者快速理解核心内容。

请阅读以下文本：

"""
5月22日，有“大模型第一股”之称的智谱（02513.HK）股价大涨，盘中一度涨超32%。
消息面上，智谱发布了GLM-5.1高速版。5月22日，智谱宣布，面向部分企业客户提供GLM-5.1高速版API“GLM-5.1-HighSpeed”，其模型输出速度达到400 Tokens/s，刷新当前全球大模型厂商API的速度上限。GLM-5.1高速版适用于AI编程、实时交互、商业决策、实时语音等对响应延迟要求极高的场景，现已面向智谱MaaS平台部分企业客户开放。
受此提振，港股人工智能应用股走强，另一大模型明星股MiniMax盘中一度涨超20%。
"""

请遵守以下要求：
1. 不要编造原文没有的信息。
2. 优先提炼与行业趋势、公司动态、技术变化、市场影响相关的信息。
3. 如果原文信息不足，请明确说明“原文未提及”。
4. 输出内容要简洁、专业、适合用于研究笔记。

请按照以下格式输出：

## 一句话总结

## 关键信息
- 
- 
- 

## 涉及主体
- 公司/机构：
- 产品/技术：
- 人物：

## 可能影响
- 对行业：
- 对公司：
- 对用户/客户：

## 不确定信息
-
```

### 测试输出

```
## 一句话总结  
智谱发布输出速度达400 Tokens/s的GLM-5.1高速版API，刷新全球大模型速度上限，推动自身股价大涨并带动港股AI应用股走强。

## 关键信息  
- 智谱（02513.HK）5月22日盘中涨超32%，收涨未提及。  
- 发布GLM-5.1高速版，模型输出速度达400 Tokens/s，刷新全球大模型厂商API速度上限。  
- 该版本面向部分企业客户开放，适用于AI编程、实时交互、商业决策、实时语音等低延迟场景。  
- 受此影响，港股人工智能应用股走强，另一大模型股MiniMax盘中涨超20%。

## 涉及主体  
- 公司/机构：智谱、MiniMax  
- 产品/技术：GLM-5.1高速版、GLM-5.1-HighSpeed API、智谱MaaS平台  
- 人物：原文未提及

## 可能影响  
- 对行业：刷新全球大模型API速度记录，可能推动行业向更高实时性、更低延迟方向竞争。  
- 对公司：智谱股价获得短期提振，同类AI应用股（如MiniMax）也受正面情绪带动。  
- 对用户/客户：部分企业客户可率先获得高速API能力，提升AI编程、实时语音等场景的响应效率。

## 不确定信息  
- 原文未提及GLM-5.1高速版的具体定价、开放范围扩大时间表、与其他主流模型（如GPT、Claude）的速度对比细节。  
- 未说明股价大涨是仅因技术发布，还是叠加其他市场因素。  
- 未提供MiniMax的具体技术动态或与智谱的直接关联。
```

### Research Prompt 2：带分类判断的行业信息总结

### 测试输入

```
你是一个 AI 行业研究助理。

请分析用户提供的行业文本，并完成“总结 + 分类 + 影响判断”。

文本如下：

"""
5月22日，有“大模型第一股”之称的智谱（02513.HK）股价大涨，盘中一度涨超32%。
消息面上，智谱发布了GLM-5.1高速版。5月22日，智谱宣布，面向部分企业客户提供GLM-5.1高速版API“GLM-5.1-HighSpeed”，其模型输出速度达到400 Tokens/s，刷新当前全球大模型厂商API的速度上限。GLM-5.1高速版适用于AI编程、实时交互、商业决策、实时语音等对响应延迟要求极高的场景，现已面向智谱MaaS平台部分企业客户开放。
受此提振，港股人工智能应用股走强，另一大模型明星股MiniMax盘中一度涨超20%。
"""

分类标签只能从以下选项中选择：
- 技术突破
- 产品发布
- 融资并购
- 政策监管
- 市场竞争
- 商业合作
- 其他

要求：
1. 只基于原文分析，不要补充外部信息。
2. 如果无法判断，请选择“其他”，并说明原因。
3. 输出要适合写入行业研究数据库。
4. 输出必须是合法 JSON，不要在 JSON 外输出额外解释。

输出格式：

{
  "summary": "一句话总结",
  "category": "分类标签",
  "key_points": ["要点1", "要点2", "要点3"],
  "entities": {
    "companies": [],
    "products": [],
    "technologies": []
  },
  "impact": {
    "industry": "对行业的影响",
    "company": "对相关公司的影响",
    "users": "对用户或客户的影响"
  },
  "uncertainties": ["不确定信息1"]
}
```

你是一个专业的 AI 行业研究分析师。

请参考示例的风格，对输入文本进行总结。

输入：
OpenAI 发布了新的企业级功能，支持团队统一管理模型调用权限和数据访问策略。

输出：

## Research Prompt 优化记录

### 已完成优化

- [x] Prompt 2：加强 JSON 解析友好性
- [x] Prompt 2：限制 `key_points` 数量
- [x] Prompt 3：避免绝对化表达

### 优化原因

Prompt 2 后续可能用于 API 调用和程序解析，因此需要保证输出为原始 JSON 对象，并控制字段内容长度。

Prompt 3 在测试中出现了略偏绝对化的表达，因此增加约束，要求模型避免使用“第一”“最强”“全球最快”等绝对化表述，除非原文明确使用。

Research 类 Prompt 已完成 D2 阶段的手动测试与轻量优化，可以作为 Nexus Research 后续摘要、分类和信息抽取能力的 Prompt 基础。
